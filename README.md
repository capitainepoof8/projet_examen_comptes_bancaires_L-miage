# objectif

Le programme doit permettre d'afficher le menu ci-dessous et de fonctionner dans une boucle continue jusqu'à ce que l'utilisateur choisisse de quitter.a note execution uniquement dans cmd
```
===== AGENCE CESAG BANK =====
1. Afficher tous les comptes
2. Consulter le solde d'un compte
3. Créer un nouveau compte
4. Effectuer un dépôt
5. Effectuer un retrait
6. Effectuer un transfert entre deux comptes
7. Quitter
==============================
```
## Scénario de test suggéré
 
1. Afficher tous les comptes (vérifier les 3 comptes de départ)
2. Créer un nouveau compte avec un solde initial de 50 000 FCFA
3. Effectuer un dépôt de 20 000 FCFA sur le compte de Moussa Diop
4. Tenter un retrait de 500 000 FCFA sur le compte de Fatou Sarr (doit être refusé)
5. Effectuer un transfert de 30 000 FCFA du compte d'Awa Ndiaye vers le nouveau compte créé
6. Saisir volontairement une lettre à la place d'un nombre (le programme ne doit pas planter)
7. Quitter via l'option 7
